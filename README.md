<h1 align="center">📱 EASY SHOPPING VIA MOBILE </h1>

Este projeto é uma landing page simples e responsiva desenvolvida para praticar HTML e CSS.
A página apresenta a interface de um aplicativo fictício de compras via celular, com um layout moderno e organizado. O objetivo foi treinar a estrutura de uma página web, estilização com CSS e adaptação para diferentes tamanhos de tela.

<br>

<h2>🚀 Tecnologias Utilizadas </h2>
<h3>- HTML5 <br> - CSS3</h3>
